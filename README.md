# Consumindo API com Axios 🚀

Projeto de estudo desenvolvido para praticar o consumo de APIs REST utilizando a biblioteca Axios em um ecossistema React.

## 📋 Sobre o Projeto

Este projeto é um caso de estudo prático que tem como objetivo principal demonstrar o consumo de dados da API pública **JSONPlaceholder**. A aplicação busca uma lista de usuários e os exibe de forma organizada em uma tabela estruturada, contando também com botões de paginação no final da listagem para navegar entre os dados.

## ⚙️ Como o Projeto Funciona

1. **Requisição de Dados:** Ao iniciar, a aplicação faz uma requisição HTTP do tipo `GET` para o endpoint de usuários da JSONPlaceholder utilizando o **Axios**.
2. **Renderização:** Os dados retornados (como Nome, Email, Empresa e Cidade) são processados e renderizados dinamicamente em uma tabela HTML.
3. **Paginação:** Na parte inferior da tabela, existem botões de paginação ("Voltar" e "Avançar") que permitem ao usuário navegar pelas páginas de registros de forma simples.
